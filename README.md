## 주요 기능

1. **업종 선택 기능**  
   - 업종 선택 시 관련 업종에 포함된 기업들을 자동으로 표시.

   - 기업을 선택하면 해당 기업의 **요약 정보**와 **홈페이지 링크**를 출력.
